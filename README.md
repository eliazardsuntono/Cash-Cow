# Cash-Cow
Budgeting App

## Important Dependencies to Download
```
https://nodejs.org/en/download/package-manager
```
## Download Expo Go on the app store!

## To Run Frontend:
```
cd cashcow
npm run
```
## If an error for too many files opens then download watchman:
```
sudo apt install watchman
``

To Run Backend:
```
node index.js
```
